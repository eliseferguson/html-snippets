Create reusable HTML/Text snippets and insert them into your posts or pages.

== Description ==

This plugin creates a custom post type for HTML Snippets.  The snippets are then available in a drop down on your posts or pages screen.

Select the appropriate snippet from the drop down and insert it at the end of your existing content.

== Installation ==

1. Upload `html-snippets.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Install via Github Updater if you want live update notices
